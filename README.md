## Parkinsons-Tremor-Detection

#### Goal: detect Parkinson's tremor in patients with Parkinson's disease, and enhance medical devices for motion disorders with AI capabilities.

#### This repo provides a machine learning model to detect and distinguish Parkinson's tremor from normal daily activities such as walking, running, standing, driving, etc.

##### See "Parkinson'sTremorDetection.ipynb" for detailed results and methods.
